================================================================================

Step 1: Database Migration (CRITICAL)
  $ mysql -u root -p < setup.sql
  Adds new columns to workers and attendance tables

Step 2: Install recharts (if not already installed)
  $ cd frontend && npm install
  Recharts already added to package.json

Step 3: Start Backend
  $ cd backend && npm run dev
  Runs on port 5000
  All 16 API endpoints ready

Step 4: Start Frontend
  $ cd frontend && npm run dev
  Runs on port 5050
  All pages with new features ready

FEATURES VERIFICATION:
================================================================================

Feature #1: Sorting by Job Type
  - WorkersPage has dropdown with all 5 job types
  - Select type to filter workers list
  - "All Job Types" option includes everyone

Feature #2: India Timezone (12-hour format)
  - All times display: 9:30 AM, 2:45 PM (not 09:30 or 14:45)
  - Server uses Asia/Kolkata timezone
  - Check-in/out times match India time, not browser timezone

Feature #3: Search Across All Pages
  - SearchBar component used in WorkersPage and DashboardPage
  - Search by name, worker_id, or phone (case-insensitive)
  - Real-time with 300ms debounce for performance

Feature #4: Dashboard Date-Wise Filtering
  - DateRangeSelector component on DashboardPage
  - Select date → Shows only workers created on that date
  - Shows their attendance records for that date

Feature #5: Absence Reason Modal
  - When changing from Present to Absent, modal appears
  - Select reason: Sick Leave, Emergency, Personal Leave, Holiday, Other
  - Reason stored in database and shown in table

Feature #6: Time Spent Display
  - After both check-in and check-out set
  - Shows "Time Spent: 08:30:45" (HH:MM:SS format)
  - No "Done" button - just time display
  - "Done" button completely removed

Feature #7: Landing Page Analytics
  - Pie chart shows today's Present vs Absent (green/red colors)
  - Bar chart shows past 7 days attendance trend
  - Summary cards: Total Workers, Present Today, Absent Today, Attendance %
  - Detailed table with all attendance records
  - All charts from recharts library (interactive)

Feature #8: Worker Management
  - Edit button: Modal to update name, phone, job_type
  - Deactivate button: Shows reason modal, then hides worker
  - Activate button: Appears for inactive workers, reactivates them
  - Inactive workers don't appear in attendance marking

Feature #9: Auto-Generate Worker IDs
  - Add Worker form: worker_id field is auto-populated
  - Shows next ID (e.g., W009) and is disabled (read-only)
  - Backend tracks worker_id_sequence for each worker
  - Format: W + 3-digit number (W001, W002, ..., W999)

Feature #10: ID Card Popup
  - Click on any worker_id in WorkersPage table
  - Opens beautiful ID card popup with:
    - Worker ID
    - Name
    - Job Type
    - Phone
    - Member Since (created date)
    - Active/Inactive status badge
  - Close with X button or click outside
  - Responsive on mobile devices
